Bella Penteadeiras - Portfólio
Este é um projeto de demonstração de um E-commerce de Penteadeiras, criado para colocar em prática meus conhecimentos de desenvolvimento Front-end.

Tecnologias Utilizadas
HTML5: Estrutura do site.
CSS3: Design, cores e estilo (layout responsivo).
JavaScript: Lógica para cálculo de preços e chatbot interativo.
